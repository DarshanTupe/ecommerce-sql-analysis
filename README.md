# Ecommerce SQL Analysis

This project is a simple SQL analysis done using PostgreSQL on an ecommerce dataset.

The goal of the project was to practice writing SQL queries and explore the data to
answer some basic business questions. During this analysis I worked with tables such
as orders, order_items, products, and customers.

Some of the things I looked at in this project:

- Revenue by product category
- Average order value
- Month-over-month revenue growth
- One-time vs repeat customers
- Revenue by customer state
- Basic RFM (Recency, Frequency, Monetary) analysis

Most of the queries used for this analysis are in the file:

ecommerce_analysis.sql

This project helped me get more comfortable with joins, CTEs, window functions,
and general SQL data analysis.
